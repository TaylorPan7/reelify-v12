Reelify API Documentation

Welcome to the Reelify API documentation. Our API enables you to programmatically access our AI-powered content creation tools.

API Availability

The API is available to users on our Pro and Enterprise plans. If you're interested in API access, please check our pricing or contact our sales team for custom enterprise solutions.

Getting Started

Authentication

All API requests require authentication using an API key. You can generate an API key in your account settings.

curl -X GET "https://api.reelify.com/v1/account" \
-H "Authorization: Bearer YOUR_API_KEY"

Base URL

All API requests should be made to the following base URL:

https://api.reelify.com/v1/

Video Generator API

Generate a Video

Create a new AI-generated video based on a text prompt.

curl -X POST https://api.reelify.com/v1/generate-video \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "prompt": "Create an engaging product demo video",
  "duration": 60,
  "style": "modern"
}'

Check Video Status

Check the status of a video generation request.

GET /videos/{video_id}/status

Retrieve Video

Get the URL to a generated video.

GET /videos/{video_id}

Image Generator API

Generate an Image

Create a new AI-generated image based on a text prompt.

POST /images/generate

{
  "prompt": "A mountain landscape at sunset",
  "style": "photorealistic",
  "resolution": "1024x1024"
}

Retrieve Image

Get the URL to a generated image.

GET /images/{image_id}

Captioner API

Generate Captions

Create captions for a video.

POST /captions/generate

{
  "video_url": "https://example.com/video.mp4",
  "language": "en",
  "format": "srt"
}

Retrieve Captions

Get the URL to generated captions.

GET /captions/{caption_id}

Rate Limits

The API enforces rate limits based on your subscription plan:

  • Pro Plan: 100 requests per minute
  • Enterprise Plan: Custom limits based on your agreement

Error Handling

The API uses standard HTTP status codes to indicate the success or failure of a request.

// Example error response
{
  "error": {
    "code": "invalid_request",
    "message": "The prompt is too long. Maximum length is 1000 characters.",
    "status": 400
  }
}

Webhooks

Set up webhooks to receive real-time notifications when your resources are created or updated.

POST /webhooks

{
  "url": "https://example.com/webhooks/reelify",
  "events": ["video.created", "image.created", "caption.created"]
}
